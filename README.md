# Ultimate React - Redux starter

Starter for your React-Redux app

### How to setup environment

```
> git clone git@github.com:henrikra/ultimate-react-redux-starter.git
> vagrant up
> vagrant ssh
> cd /vagrant
> npm i
> npm start
```

### Access the app
[http://localhost:8080](http://localhost:8080)

### How to run tests
`npm run test`
